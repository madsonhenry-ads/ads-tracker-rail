# Research Sources & Search Strategies

## Primary Research Sources

### YouTube Channels & Content Types
- **Tool Tutorials:** Search for "[Tool Name] tutorial 2026" or "[Tool Name] review"
- **Case Studies:** Search for "cloaking case study" or "Facebook ads fraud detection"
- **Expert Interviews:** Search for "affiliate marketing expert" or "ad fraud specialist"
- **Technical Breakdowns:** Search for "how cloaking works" or "Facebook ads detection"

**Recommended Search Queries:**
```
- "The White Rabbit cloaking tutorial"
- "Facebook ads spy tools comparison 2026"
- "How affiliates bypass Facebook moderation"
- "Cloaking detection methods"
- "Ad fraud case study"
```

## Search Strategy Framework

### Broad Research Phase
**Goal:** Understand landscape and identify key players

**Queries:**
```
- "[Topic] overview 2026"
- "[Topic] tutorial for beginners"
- "[Topic] comparison"
- "best [topic] tools"
```

**Sources:** General web search, YouTube, blog posts

### Deep Dive Phase
**Goal:** Understand specific tools and techniques

**Queries:**
```
- "[Tool Name] features"
- "[Tool Name] pricing"
- "[Tool Name] review"
- "[Tool Name] vs [Competitor]"
- "how to use [Tool Name]"
```

**Sources:** Official documentation, YouTube tutorials, Reddit discussions

### Community Insights Phase
**Goal:** Understand real-world usage and experiences

**Queries:**
```
- site:reddit.com "[topic]"
- site:blackhatworld.com "[topic]"
- site:afflift.com "[topic]"
- "[topic] forum discussion"
```

**Sources:** Reddit, forums, community discussions

### Advanced Techniques Phase
**Goal:** Understand sophisticated methods and edge cases

**Queries:**
```
- "[topic] advanced techniques"
- "[topic] bypass methods"
- "[topic] detection evasion"
- "[topic] case study"
```

**Sources:** Forums, expert blogs, technical documentation

## Quality Assurance Checklist

When compiling research:

- [ ] Each major claim has at least 2 sources
- [ ] All sources are cited with URLs
- [ ] Information is current (within 6 months)
- [ ] Conflicting information is noted
- [ ] Technical details are verified
- [ ] Pricing information is up-to-date
- [ ] Examples are realistic and relevant
- [ ] No plagiarism - all content is paraphrased or quoted with attribution

## Common Research Pitfalls to Avoid

1. **Outdated Information** - Always check publication dates
2. **Single Source Reliance** - Verify with multiple sources
3. **Biased Sources** - Consider tool creators' marketing bias
4. **Incomplete Data** - Seek multiple perspectives
5. **Unverified Claims** - Don't repeat unsubstantiated claims
6. **Missing Context** - Understand the full situation before concluding
7. **Rapid Changes** - Note that tools and techniques evolve quickly
